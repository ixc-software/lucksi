NAME

ntohl - convert a 32 bit value from network-byte order to host-byte order.  

SYNOPSIS

#include <lwip/inet.h>

u32_t ntohl(u_32t hostlong);  

DESCRIPTION

The ntohl function converts a 32-bit value from network-byte order to host-byte order. Typically this function is used with internet address and ports.  

RETURN VALUE

The 32-bit value in host-byte order.
