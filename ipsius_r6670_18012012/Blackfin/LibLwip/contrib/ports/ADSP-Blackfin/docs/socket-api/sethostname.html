NAME

gethostname, sethostname - get/set host name  

SYNOPSIS

#include <unistd.h>

int gethostname(char *name, size_t len);
int sethostname(const char *name, size_t len);  

DESCRIPTION

These functions are used to access or to change the host name of the current processor. Note: This function is NOT supported by LwIP yet.  

RETURN VALUE

On success, zero is returned. On error, -1 is returned, and errno is set appropriately.  

ERRORS

EINVAL
len is negative or, for sethostname, len is larger than the maximum allowed size.
EFAULT
name is an invalid address. Not Supported
