NAME

close - closes the socket associated with the descriptor.  

SYNOPSIS

#include <lwip/sockets.h>

int close(int s);  

DESCRIPTION

Closes the socket associated with the passed socket descriptor s.The passed socket descriptor must be returned by the socket call. If data is still present then close tries send the data on the socket before closing the connection.  

RETURN VALUE

On success, zero is returned. On error, -1 is returned, and the specific error number can be obtained by calling the function GetSocketError. In a multi-threaded environment the last error number is held seperately for each thread.  

ERRORS

EINVAL
The argument s is not a valid descriptor.
