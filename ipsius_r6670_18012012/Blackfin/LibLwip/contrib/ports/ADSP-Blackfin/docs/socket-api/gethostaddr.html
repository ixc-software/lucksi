NAME

gethostaddr - gets the host address string in dot decimal notation.  

SYNOPSIS

#include <lwip/inet.h>

int gethostaddr(int nfce_no, char* host_addr);  

DESCRIPTION

gethostaddr could be used to get the IP address string in dot decimal notation. The parameter nfce_no identifies the interface number.host_addr is a buffer pointer to hold the address. It must be atleast 16 bytes long.Default network interface number is 0. Applications can use this function to retrieve the host ip address once it obtains from DHCP server.  

RETURN VALUE

The call returns -1 if it fails. Returns 1 upon success and host_addr contains the IP address string in dot decimal notation.
