NAME

inet_ntoa - converts the supplied numeric internet address in to a string that represents the internet address in dot format.  

SYNOPSIS

#include <lwip/inet.h>

char* inet_ntoa(u32_t hostaddr);  

DESCRIPTION

converts the supplied numeric internet address in to a string that represents the internet address in dot format.  

RETURN VALUE

Internet address in dot format.
