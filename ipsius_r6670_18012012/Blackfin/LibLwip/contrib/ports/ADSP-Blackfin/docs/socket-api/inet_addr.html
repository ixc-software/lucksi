NAME

inet_addr - converts the given standard IPv4 dotted decimal notation of ip address in to an integer value suitable for use as an internet address.  

SYNOPSIS

#include <lwip/inet.h>

u32_t inet_addr(const char * cp);  

DESCRIPTION

converts the given standard IPv4 dotted decimal notation of ip address in to an integer value suitable for use as an internet address.  

RETURN VALUE

Integer value of the internet address.
