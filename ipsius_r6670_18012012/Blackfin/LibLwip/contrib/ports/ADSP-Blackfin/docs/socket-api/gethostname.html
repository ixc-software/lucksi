NAME

gethostname, sethostname - get/set host name  

SYNOPSIS

#include <lwip/sockets.h>

int gethostname(char *name, size_t len);
int sethostname(const char *name, size_t len);  

DESCRIPTION

These functions are used to access or to change the host name of the current processor. Note: This function is NOT supported by LwIP yet.  

RETURN VALUE

On success, zero is returned. On error, -1 is returned, and error value is obtained by calling the function GetSocketError(). In multithreaded environments last error number is held seperately for each thread.  

ERRORS

EINVAL
len is invalid.
EFAULT
name is an invalid address.
