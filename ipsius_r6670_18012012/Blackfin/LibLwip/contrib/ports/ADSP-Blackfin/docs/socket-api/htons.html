NAME

htons - convert a 16 bit value from host-byte order to network-byte order.  

SYNOPSIS

#include <lwip/inet.h>

u16_t htons(u_16t hostshort);  

DESCRIPTION

The htons function converts a 16-bit value from host-byte order to network-byte order. Typically this function is used with internet address and ports.  

RETURN VALUE

The 16-bit value in host-byte order.
