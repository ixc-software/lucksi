NAME

ntohs - convert a 16 bit value from network-byte order to host-byte order.  

SYNOPSIS

#include <lwip/inet.h>

u16_t ntohs(u_16t hostshort);  

DESCRIPTION

The htons function converts a 16-bit value from network-byte order to host-byte order. Typically this function is used with internet address and ports.  

RETURN VALUE

The 16-bit value in host-byte order.
