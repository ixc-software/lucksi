

1. Платформо зависимая фунция сбора информации

    bool PlatformProt::GetMachineID(const std::string &info);
    
    На Linux вычленить часть данный из dmidecode (CPU? Motherboard?)    
    
    На Windows получить серийный номер тома с ОС
    
2. Сбор инфы о машине пользователя 

    NObjMachineID.Get()
    
    получется данные от GetMachineID()
    применяет строке хэш (например, MD5 + base64)
    сохраняет результат в pcid.txt     

    Файл передается для генерации сертификата.
    
    
3. Генерация сертификата

    прочитать данные из pcid.txt в строку
    вызывать функции string MakeSerial(string &id)
    результат записать в файл serial.txt
    
    отослать файл пользователю
    
    код генерации не должен быть включен в проект!!

4. Проверка сертификата во время работы 
    
    прочитать serial.txt в строку
    вызывать функцию bool VerifySerial(GetMachineID(), string &serial)
    при false выйти из программы
